CdiTools
=======

**Que es CdiTools**

CdiTools es un modulo con multiples herramientas


Instalacion
============

Installation via composer, agregar a composer.json:
```
"require" : {
    "cdi/cditools": "dev-master"
}
```



```
'modules' => array(
    'Application',
    'CdiTaools'             
)
```
